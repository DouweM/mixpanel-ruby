config_http(&block) click to toggle source

Ruby's default SSL does not verify the server certificate. To verify a certificate, or install a proxy, pass a block to Mixpanel::use_ssl that configures the Net::HTTP object. For example, if running in Ubuntu Linux, you can run

Mixpanel::use_ssl do |http|
    http.ca_path = '/etc/ssl/certs'
    http.ca_file = '/etc/ssl/certs/ca-certificates.crt'
    http.verify_mode = OpenSSL::SSL::VERIFY_PEER
end

Mixpanel Consumer and BufferedConsumer will call your block to configure their connections

# File lib/mixpanel-ruby/consumer.rb, line 23
def self.config_http(&block)
  @@init_http = block
end
